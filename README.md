# Student Stress Level and Academic Performance Analysis

This project analyzes the relationship between students' mental health indicators

(stress, anxiety, depression, panic attacks) and academic performance.

# Project Structure

- data/: dataset files

- notebooks/: exploratory analysis and experiments

- scripts/: reusable Python scripts

- figures/: generated plots and figures



# How to Run

1. Install Python 3.8+

2. Install dependencies:

&nbsp;  pip install -r requirements.txt

3. Run notebooks in order or execute scripts in the scripts folder.
